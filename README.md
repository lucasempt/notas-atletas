# notas-atletas
Projeto: Notas dos Atletas

Este projeto foi desenvolvido como parte do programa DEVstart e tem como objetivo criar uma aplicação em JavaScript capaz de:

1. Receber o nome e cinco notas de atletas participantes de uma competição de ginástica artística.
2. Desconsiderar a maior e a menor nota.
3. Calcular a média das três notas centrais.
4. Exibir para o usuário o nome do atleta, as notas obtidas e a média válida.
